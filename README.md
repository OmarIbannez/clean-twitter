# Installation
1. Clone the repo `git clone git@github.com:OmarIbannez/clean-twitter.git`.
2. Create a [Twitter App](https://developer.twitter.com/en/apps) to get the credentials needed.
4. Set the credential on the top of the script (`clean.py`).
5. Install the dependencies on the `requirements.txt` file.
5. Success.
